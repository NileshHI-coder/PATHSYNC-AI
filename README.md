# PathSync AI 🚀

## AI-Powered Career Navigation Platform

PathSync AI is an intelligent education-to-employment guidance platform designed to help students identify career pathways, analyze skill gaps, receive personalized mentorship, and improve employability readiness.

Built for the **Higher Education & Skilling** hackathon track, the platform focuses on solving fragmented education-to-career transitions using AI-powered guidance tools.

---

# 🌟 Problem Statement

Students often face fragmented journeys across:

* Career guidance
* Skill development
* Industry readiness
* Learning resources
* Employment pathways

Many learners especially from rural, low-income, and first-generation backgrounds lack access to:

* Personalized mentorship
* Career clarity
* Skill gap awareness
* Structured roadmaps
* Real-world readiness

PathSync AI aims to bridge this gap using AI-powered career navigation.

# 🤖 AI Methodology

The platform uses Groq-powered Llama AI models to provide:

* Personalized responses
* Career guidance
* Skill recommendations
* Learning roadmaps
* Student mentorship

### AI Workflow

1. User submits query
2. Query sent to Groq API
3. Llama model processes request
4. AI generates career-focused response
5. Response displayed in chatbot UI

## 🔐 API Key Security Note

For security reasons, the original Groq API key used during development and demo testing has been removed from this public repository before submission.

To run the AI chatbot locally:

1. Create your own Groq API key from:
   https://console.groq.com/keys

2. Add your key in the project configuration file:

const API_KEY = "YOUR_GROQ_API_KEY";

3. Run the project locally.

The demo video submitted for the hackathon shows the fully working AI-powered chatbot integrated using Groq API (Llama 3).
